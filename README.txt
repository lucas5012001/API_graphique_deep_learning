Ce dossier contient deux API permettant chacune de donner accès à un modèle de deep Learning à l'aide d'une petite interface graphique.

La première se lance avec app-DETR.py et permet de charger une photo, puis de récupérer la photo annotée par un modèle
de deep Learning de type DETR.

La seconde se lance avec app-mp3.py et permet de charger un fichier mp3 (en français), puis de récupérer la transcription écrite des paroles enregistrées dedans. Cela est fait en utilisant le modèle de deep learning wav2vec2.
